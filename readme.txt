Project.
This project aims to detect SNP occurances in .aln files, so that the pathogenicity of them may be rated.

Prerequisites.
Python3.10+ with packages listed below.
Biopython
Biopython: AlignIO capabilities

Usage.
To run the program, follow the steps below.
1. Insert the following into your CLI:
python3 snp.py [path to .aln file]

Example.
python3 snp.py .\POU5F1.aln
